# chess-bot
This is a chess bot trained by neural network with high rated blitz games
built with TensorFlow, and Pygame

it currently plays at a lower Elo and is not trained to punish player misplays

### Install Dependencies:
```bash
pip install numpy pygame tensorflow python-chess
```

### Run the main script to play:
```bash
python play_game.py
```
